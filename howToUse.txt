


	BusTagReader : 버스 데이터 정산 프로그램


	제작 :

	2020 김진환


	주의 : 

	1. 버스데이터추출 폴더 자체를 옮기는 것은 상관없으나 포함된 자료의 이름 또는 위치가 변경 되어선 안된다.

	(내부 파일과 폴더 명) 검색자료, BusTagReader.exe, 버스직원데이터.xls, 요금정보.txt


	2. 조회는 한번만 가능하다. 조회가 한번 종료되고 다시 다른 조회를 하고 싶다면 프로그램을 재실행해야 한다.

	3. 요금 정보는 '요금정보.txt'에서 변경 가능하다. 

	평택, 교대, 사당, 천안 순서로 해당 금액을 빈칸 없이 쉼표로 구분한다.  (ex) 500,1500,1500,500


	사용 방법 :

	1. 버스직원데이터.xls에 직원 데이터를 입력해야한다.

	2. 버스 리더기에서 데이터 파일을 꺼내와 프로그램의 '파일 열기'로 등록한다.

 	- 평택, 교대, 사당, 천안 순으로 GLOG_001.TXT ~ GLOG_004.TXT 파일이다.

	- 반드시 모든 파일을 등록할 필요없다. 조회가 필요한 버스 데이터 파일만을 등록한다.

	3. 조회 시작 일자와 종료 일자를 입력한다.

	- 형식은 2020-12-01으로 -을 이용하여 년,월,일을 구분한다.

	4. 상단의 Run -> 실행을 클릭한다.

 	5. 검색된 자료의 수가 출력되었다면 조회가 완료된 것이다.

 	6. '검색자료' 폴더에서 조회한 날짜, 시간으로 명명된 엑셀 파일에 조회 결과가 저장되어있다.
